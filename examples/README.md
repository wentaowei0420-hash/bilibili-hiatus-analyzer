# 📝 示例文件

这个目录包含了B站催更分析器的示例和配置文件。

## 📁 文件列表

### 1. config_example.txt

**Cookie配置示例文件**

这个文件提供了详细的Cookie配置说明：
- 如何从浏览器获取Cookie
- Cookie的格式示例
- 配置步骤说明
- 常见问题提示

**用途**：帮助新手用户正确配置Cookie

### 2. example_output.csv

**输出文件示例**

这是脚本运行后生成的CSV文件示例，包含：
- UP主姓名
- UP主UID
- 最新视频标题
- 发布日期
- 未更新天数
- 播放量
- 视频链接

**用途**：让用户了解脚本的输出格式

## 📖 如何使用

### Cookie配置

1. 打开 `config_example.txt`
2. 按照说明获取你的Cookie
3. 在主程序 `bilibili_hiatus_analyzer.py` 中配置

### 查看输出示例

直接打开 `example_output.csv`：
- 用Excel打开
- 用Google Sheets打开
- 用任何文本编辑器打开

## 💡 提示

- ⚠️ **不要**将包含真实Cookie的文件提交到Git
- ⚠️ **不要**分享包含个人数据的CSV文件
- ✅ 这些示例文件是安全的，不包含真实数据
